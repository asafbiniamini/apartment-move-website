Create 360° Panorama

Step 1: Launch Sweet Home 3D

Click the button below to open Sweet Home 3D on your computer:

Step 2: Design Your Room

  1. Draw your room plan using the wall tools
  2. Add furniture from the catalog
  3. Set up lighting and textures
  4. Position the camera where you want the panorama

Step 3: Create 360° Panorama

  1. Go to File → Export → 360° Panorama
  2. Choose your desired settings (resolution, format)
  3. Save the panorama image
  4. Upload it to this website using "Upload Pictures"

Step 4: View in 360°

Once uploaded, use the "View Room" button and switch to "360°" mode to see your panorama!

Tips for Better Panoramas

  • High Resolution: Use at least 2048x1024 for good quality
  • Camera Position: Place camera at eye level (1.6m height)
  • Lighting: Add realistic lighting for better results
  • Furniture: Include all planned furniture for accurate planning

Sweet Home 3D Quick Guide

Getting Started

1. Create New Project

File → New → Choose your room dimensions

2. Draw Walls

Click "Wall" tool → Click to start wall → Click to end wall → Press ESC

3. Add Doors & Windows

Click "Door" or "Window" tool → Click on wall where you want it

Adding Furniture

1. Open Catalog

View → Catalog (or press F2)

2. Browse Categories

Kitchen → Appliances, Bathroom → Fixtures, etc.

3. Add to Room

Drag item from catalog to room → Use handles to resize/rotate

Creating 360° Panoramas

1. Switch to 3D View

View → 3D View (or press F3)

2. Position Camera

Right-click and drag to move camera to center of room

3. Export Panorama

File → Export → Image → Check "Panorama" → Set resolution → Save

Pro Tips

  • Undo/Redo: Ctrl+Z / Ctrl+Y
  • Save Often: Ctrl+S
  • Zoom: Mouse wheel or Ctrl + mouse wheel
  • Pan: Middle mouse button + drag
  • Rotate 3D: Right mouse button + drag

Troubleshooting

  • Software won't start: Run in terminal: sweethome3d
  • Can't find panorama export: Try File → Export → Image → Look for "Panorama" checkbox
  • Furniture won't place: Make sure you're in "Furniture" mode (not "Wall" mode)
